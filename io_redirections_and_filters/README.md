# Shell, I/O Redirections and Filters

This project introduces shell input/output redirections and filters in Bash scripting.  
All scripts are tested on Ubuntu 20.04 LTS and must follow strict format rules.

---

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- All files must be executable
- First line of all scripts: `#!/bin/bash`
- All files should be exactly two lines long
- No use of backticks, `&&`, `||`, `;`, `sed`, or `awk`
- Each file must end with a new line

---

## Scripts Description

### 0. Hello World
**File:** `0-hello_world`  
Prints “Hello, World” followed by a new line to the standard output.

**Example:**
```bash
$ ./0-hello_world
Hello, World
